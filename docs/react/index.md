## なぜReactでJSX?

### そもそもJSXとは?

JavaScript Syntax Extension　or JavaScript XMLともいわれる。
JavaScript の中に XML ライクな記述ができるようにした構文拡張(言語標準には含まれない特殊な用途のための便利な構文を、 後づけで使えるようにしたもの)

### JSXとコンパイル

JSX は標準の JavaScript にコンパイルされることを前提としてる。
